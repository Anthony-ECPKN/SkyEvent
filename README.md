# SkyEvent
This is a code and dataset base for event camera in large-scale 3D reconstruction

# 无人机多模态数据集采集记录

## 数据集概览

本数据集目前包含3天共7次飞行任务采集的多模态传感器数据，总数据量达437.22GB。数据包括事件相机流和RGB视频流，均在70-100米高度采集，涵盖不同光照条件和飞行模式。

## 文件格式说明

- **Event数据**: 
  - `.raw` - 事件相机原始数据文件
  - `.bias` - 事件相机偏置文件
  - `.tmp_index` - 临时索引文件

- **RGB数据**:
  - `.MP4` - 视频文件
  - `.LRF` - 低分辨率预览文件（由DJI设备生成）





## 数据集特点

1. **多高度采集**: 数据在70米、80米和98米三种高度采集
2. **多飞行模式**: 包含手动绕飞、自动扫描航线和多方向采样等多种飞行模式
3. **多光照条件**: 涵盖午后、黄昏等不同光照条件
4. **时空对齐**: 事件流和RGB流保持时间同步
5. **丰富场景**: 包含建筑区域和开阔场地等多种场景

## 使用建议

1. 事件相机数据需要使用专用软件（如Prophesee提供的工具）进行解析和处理
2. RGB视频文件可直接使用常规媒体播放器播放
